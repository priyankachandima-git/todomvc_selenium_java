# Selenium Java Test Automation Framework

This project is an automation framework built to test [ToDOMVC](https://todomvc.com/examples/react/dist/) website using Selenium WebDriver, Java, and TestNG. The framework provides a robust testing infrastructure to ensure the functionality and stability of the ToDOMVC application.

## Tech Stack

- **Testing Framework:** Selenium WebDriver + TestNG
- **Programming Language:** Java

## Features

- **End-to-End Testing:** Comprehensive testing suite covering various aspects of the ToDOMVC application.
- **Modular Structure:** Organized directory structure for easy maintenance and scalability.
- **Custom Utilities:** Utilizes custom utilities for common automation tasks and actions.
- **Reporting:** Integrated with Surefire Reports for detailed test reports.
- **CI/CD Integration:** Configured with GitHub Actions for continuous integration and automated testing.

## Getting Started

To get started with the project, follow these steps:

1. **Clone the Repository:**
   git clone https://github.com/your-username/todomvc-automation.git
   cd todomvc-automation
   
2. **Install dependencies:**
   mvn clean install
   
3. **Clone the Repository:**
   mvn test
